### ✅ Proper Way to Close eclipse & server

1️⃣ **Stop Server**
Servers tab → Right click Tomcat → **Stop** → wait until `[Stopped]`

2️⃣ **Close Eclipse Normally**
`File → Exit` (don’t force close)

3️⃣ **Don’t run `startup.bat` manually**

###  Next Time

Open Eclipse → Servers tab → **Start** → Done ✅
