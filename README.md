## 💻 Introduction to Advanced Java

**Advanced Java is a part of Java programming** used to develop web-based and enterprise applications. It focuses on server-side programming, database connectivity, and dynamic web content. Advanced Java is mainly used to build large-scale, distributed applications.

It is based on core concepts of **Java**, which was developed by **James Gosling** in 1995 at **Sun Microsystems**. Advanced Java extends Core Java by using technologies for web and enterprise development.

### Features of Advanced Java:

* Database connectivity (JDBC)
* Server-side programming
* Supports web technologies
* Platform independent
* Enterprise-level application support

### Uses of Advanced Java:

* Web applications
* Enterprise applications
* E-commerce websites
* Banking and business software   <br><br>
